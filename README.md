# Рекомендательная система для энергосбытовой компании

## Структура проекта
- data/ – исходные EXCEL-файлы, преобразованные к формату CSV (01–14)
- models/ – обученная модель LightGBM и артефакты
- run.py – главный интерфейс командной строки
- explain.py – персональное объяснение рекомендации
- report.py – сводный отчёт о результатах

## Требования
- Python 3.13+
- Библиотеки: pandas, numpy, lightgbm, scikit-learn, shap, openpyxl (для конвертации Excel)
- Установка: `pip install -r requirements.txt`

## Запуск

### 1. Подготовка данных
Поместите исходные файлы (01–14) в папку data. Если они в формате Excel, автоматически конвертируются в CSV.

### 2. Обучение модели
Если есть подозрения, что текущая модель перестала адекватно оценивать ситуацию, запустите однократно, 
дав как можно больше данных в таком же формате, как были исходные
```
python run.py --mode train
```
Будут созданы: training_data.csv, обученная модель uplift_model.pkl.

### 3. Получение рекомендаций
```
python run.py --mode infer
```
Результат: data/recommendations_march2026.csv — наиболее вероятные рекомендации, 
предполагаемая сумма возврата долга и необходимая мера.

### 4. Объяснение рекомендации
```
python explain.py 123456
```
Где 123456 – номер лицевого счёта.

### 5. Сводный отчёт
```
python run.py --mode report
```

## Особенности решения
- **Сегментация**: кластеры должников на основе их профиля, истории оплат и реакции на меры.
- **Прогноз ожидаемого возврата**: регрессионная модель LightGBM предсказывает конкретную сумму снижения долга для каждой допустимой меры.
- **Оптимальное назначение мер**: жадный алгоритм распределяет меры с учётом месячных лимитов, максимизируя суммарный возврат.
- **Объяснимость**: SHAP-анализ показывает, какие факторы повлияли на решение по каждому должнику.
